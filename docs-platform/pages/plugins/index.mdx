## Using Plugins

### Basic Usage

All commands use the configured prefix (default: `.`):

```
.ping                    # Check bot status
.help                    # List all commands
.help <command>          # Get help for specific command
```

### Command Patterns

Plugins may have multiple aliases:

```
.translate | .tr | .tl   # Translation plugin
.youtube | .yt | .ytdl   # YouTube downloader
.sticker | .s | .stick   # Sticker maker
```

### Reply-Based Commands

Some commands work by replying to messages:

```
Reply to an image:
.sticker                 # Convert to sticker

Reply to a message:
.translate es            # Translate to Spanish
```

### With Arguments

Most plugins accept arguments:

```
.weather London          # Get weather for London
.youtube <url>           # Download YouTube video
.translate fr Hello      # Translate to French
```

## Plugin Management

### Enabling/Disabling Plugins

Plugins can be enabled or disabled by moving them in/out of the `plugins` folder or by renaming them with a `.disabled` extension:

```bash
# Disable a plugin
mv plugins/plugin.js plugins/plugin.js.disabled

# Enable a plugin
mv plugins/plugin.js.disabled plugins/plugin.js
```

### Plugin Updates

Plugins are updated with bot updates:

```bash
git pull origin main
yarn install
```

Create your own plugins! See our [Plugin Creation Guide](/plugins/creation).

## API Keys Required

Some plugins require API keys:

| Plugin | API Key | Get It From |
|--------|---------|-------------|
| ChatGPT | `OPENAI_API_KEY` | [platform.openai.com](https://platform.openai.com) |
| Gemini | `GEMINI_API_KEY` | [makersuite.google.com](https://makersuite.google.com) |
| Weather | `WEATHER_API_KEY` | [weatherapi.com](https://weatherapi.com) |
| News | `NEWS_API_KEY` | [newsapi.org](https://newsapi.org) |

Configure API keys in your `config.env` file.
